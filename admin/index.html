Google Analytics

Google Analytics is Ready

The GA4 tracking snippet is installed on all pages. To start collecting data:

  1. Go to analytics.google.com
  2. Create a new GA4 property for tmjoineryltd.co.uk
  3. Copy your Measurement ID (starts with G-)
  4. Enter it below to activate tracking

Once saved, the tracking code on your site will start sending data to your GA4 property.

Google Search Console

Search Console Setup

Google Search Console shows you how your site appears in Google Search results, which keywords bring visitors, and any indexing issues.

  1. Go to Google Search Console
  2. Add property → choose URL prefix
  3. Enter https://www.tmjoineryltd.co.uk
